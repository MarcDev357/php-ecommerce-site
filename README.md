# PHP & MySQL E-Commerce Site

A dynamic e-commerce website built with PHP and MySQL 
following MVC (Model-View-Controller) architecture.

## Features
- Product listings with full CRUD operations
- User account registration and login
- Shopping cart functionality
- Admin panel for product and order management
- User authentication and session management
- Responsive front-end built with HTML and CSS

## Tech Stack
- PHP (Backend)
- MySQL (Database)
- MVC Architecture
- HTML / CSS
- User Authentication & Session Management

## How to Run
1. Clone the repo
2. Import the SQL database file into MySQL
3. Update database credentials in config file
4. Run with XAMPP or any PHP local server
5. Visit localhost in your browser

## Project Structure
```
├── controllers/    # Application logic
├── models/         # Database interactions
├── views/          # HTML templates
├── config/         # Database configuration
└── public/         # CSS, JS, images
```
